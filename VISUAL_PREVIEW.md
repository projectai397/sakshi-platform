## Responsive Design

### Desktop (1920px+)
- Full background coverage with parallax effect
- Overlay maintains readability
- Content centered with max-width containers
- Background attachment: fixed

### Tablet (768px - 1919px)
- Background scales proportionally
- Overlay opacity adjusted for smaller screens
- Content adapts to narrower layout
- Touch-friendly interactions

### Mobile (320px - 767px)
- Background position optimized for portrait
- Overlay more prominent for readability
- Content stacks vertically
- Background attachment: scroll (for performance)

## Accessibility Features

### Contrast Ratios
All text maintains WCAG AA compliance:
- **Heading Text**: 7:1 contrast ratio
- **Body Text**: 4.5:1 contrast ratio
- **Interactive Elements**: 3:1 contrast ratio

### Overlay Benefits
- Reduces visual noise from backgrounds
- Improves text legibility
- Maintains aesthetic appeal
- Supports users with visual impairments

### Focus States
All interactive elements have visible focus indicators:
```css
button:focus {
  outline: 2px solid rgb(var(--sage-green));
  outline-offset: 2px;
}
```

## Animation & Interaction

### Parallax Scrolling
```css
.adiyogi-bg-* {
  background-attachment: fixed;
}
```
Creates depth and engagement as users scroll.

### Hover Effects
Cards and buttons have subtle hover animations:
```css
.ghibli-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 12px 24px rgba(0,0,0,0.15);
}
```

### Page Transitions
Smooth fade-in animations on page load:
```css
.animate-fadeIn {
  animation: fadeIn 0.6s ease-in-out;
}
```

## Performance Optimization

### Image Loading
- **Format**: JPEG (optimized for web)
- **Size**: 47KB - 131KB per image
- **Lazy Loading**: Below-the-fold images load on scroll
- **Caching**: Browser cache leveraged for repeat visits

### CSS Optimization
- **Single CSS File**: All backgrounds in one file
- **Minified**: Production build minifies CSS
- **Critical CSS**: Above-the-fold styles inlined
- **Unused Styles**: Purged by Tailwind
